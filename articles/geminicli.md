# Gemini CLI Overview

## Installation
*The following steps for Windows and Linux are recommended to be performed as administrator*
### Windows Installation
1. Make sure the setting, "Allow local Powershell scripts to run without being signed" is ENABLED
2. [Choclatey](https://chocolatey.org/install) must be installed.
4. Install npm via Choclatey with the following command:
```
choco install -y --force nodejs-lts
```
4. Now install [Gemini CLI](https://geminicli.com/) with the following command:
```
npm install -g @google/gemini-cli
```
### Linux Installation
1. Update repositories:
```
sudo apt update
```
2. Install nodeJS and npm:
```
sudo apt install nodejs npm
```
3. Install Gemini CLI with the following command:
```
npm install -g @google/gemini-cli
```
